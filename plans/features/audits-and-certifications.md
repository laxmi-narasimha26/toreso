# Feature — Audits & Certifications (Spec: Goals 6, 7, 8, 10)

Scope
- Manage supplier audits, reports, certificates; reminders and scheduling

Capabilities
- Panel auditor assignments; audit workflows; expiring certificate reminders; report repository

Acceptance Criteria
- Orders blocked when supplier lacks required certification for product/category/plant
- Renewal reminders issued N days before expiration

Dependencies
- RBAC (auditor), notifications, doc store, scheduling
