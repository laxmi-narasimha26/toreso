# Page Specification: Home Page

## URL
`/` or `/home`

## Page Type
- Public (unauthenticated)
- Server-Side Rendered (SSR) for SEO
- Dynamic content with caching

---

## Page Purpose

The homepage is the main entry point for new visitors, serving to:
1. Explain what Toreso is and its value proposition
2. Drive user registrations (buyers and suppliers)
3. Showcase key features and benefits
4. Build trust through social proof
5. Guide users to relevant sections
6. Optimize for SEO (primary keywords)

---

## Performance Optimization

- Critical CSS inline
- Lazy load below-fold images
- Defer non-critical JavaScript
- Optimize images (WebP with fallback)
- Minify CSS/JS
- CDN for static assets
- Server-side cache (5 minutes)
- Prefetch key pages on hover

**Target Metrics**:
- Lighthouse Score: 95+
- First Contentful Paint: <1.5s
- Largest Contentful Paint: <2.5s
- Time to Interactive: <3.5s

## Analytics Events

- `page_view` - Homepage viewed
- `search_initiated` - Search bar used
- `cta_clicked` - Any CTA button clicked
  - Button location
  - Button text
  - User type (new/returning)
- `category_viewed` - Category card clicked
- `supplier_profile_viewed` - Supplier card clicked
- `testimonial_interacted` - Testimonial carousel used
- `newsletter_subscribed` - Email submitted
- `social_clicked` - Social media link clicked
- `scroll_depth` - 25%, 50%, 75%, 100%
- `time_on_page` - Duration tracking
